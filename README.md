Analisis perilaku transaksi dan pola aktivitas keuangan untuk keperluan pengumpulan submission akhir pada modul Dicoding. Dataset ini mencakup 2.512 sampel data transaksi, yang mencakup berbagai atribut transaksi, demografi nasabah, dan pola penggunaan. Setiap entri memberikan wawasan komprehensif terhadap perilaku transaksi, memungkinkan analisis untuk keamanan finansial dan pengembangan model prediktif. Saya membuat model clustering yang selanjutnya akan digunakan untuk membuat model klasifikasi.
