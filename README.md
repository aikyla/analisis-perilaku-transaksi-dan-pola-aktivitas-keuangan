# analisis-perilaku-transaksi-dan-pola-aktivitas-keuangan
Analisis perilaku transaksi dan pola aktivitas keuangan untuk keperluan pengumpulan submission akhir pada modul Dicoding. Dataset ini mencakup 2.512 sampel data transaksi, yang mencakup berbagai atribut transaksi, demografi nasabah, dan pola penggunaan.
